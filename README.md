# 📈 MACD Strategy Backtester

Backtester de estrategia MACD sobre datos históricos reales. **Coste 0.**

## 🚀 Deploy en Streamlit Cloud (gratis)

1. Sube esta carpeta a un repo de GitHub
2. Ve a [share.streamlit.io](https://share.streamlit.io)
3. Conecta tu GitHub y selecciona el repo
4. En **Main file path** pon: `app.py`
5. ¡Listo! Tu app estará online en minutos

## 💻 Ejecución local

```bash
pip install -r requirements.txt
streamlit run app.py
```

## ⚙️ Funcionalidades

- 📊 Gráfico de velas + señales de compra/venta
- 📉 Panel MACD con histograma y líneas
- 💹 Equity curve de tu cartera
- 📋 Registro completo de operaciones
- 🔧 Parámetros ajustables: ticker, capital, fechas, EMA fast/slow/signal
